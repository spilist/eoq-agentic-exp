# 성공 기준 (Checklist)
1. **감정의 구체화:** 사용자가 말한 "사정한다/빈다"는 감정적 표현을 '우선순위 조정(Prioritization)', '스펙 아웃(Spec-out)', '리소스 재배치' 같은 구체적인 업무 용어로 변환했는가?
2. **준비 사항 제시:** 단순히 말하러 가기 전에, '버그 리포트 현황'이나 '잔여 티켓 수' 같은 데이터를 지참하라고 조언했는가?
3. **SMART 평가:** 사용자의 행동계획에 대해 'Specific(구체성)' 항목 점수를 낮게(1~2점) 책정하고, 그 이유를 명확히 설명했는가?
